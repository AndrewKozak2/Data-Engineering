# Лабораторна робота №4 - Kafka Streams

Цей проєкт реалізує обробку потокових даних з поїздок Divvy Trips за допомогою Kafka Streams.

## Вимоги:
- Java 17+ (в Docker контейнері)
- Docker та Docker Compose

## Обчислення:
1. **Середня тривалість та кількість поїздок на день**: Результат у топіку `trip-stats-daily`.
2. **Найпопулярніша початкова станція**: Результат у топіку `popular-start-station`.
3. **Топ-3 станції (старт + фініш)**: Результат у топіку `top-stations`.

## Як запустити:
1. Перейдіть до папки `lab4`.
2. Запустіть стек:
   ```bash
   docker compose up --build
   ```
3. Перевірте результати в Kafka UI: `http://localhost:8080`.

## Примітка:
Процесор очікує дані в топіку `Topic1`, які відправляє `python-producer`.
